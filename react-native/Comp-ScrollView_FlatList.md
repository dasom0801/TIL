# ScrollView 와 FlatList

|        | ScrollView                              | FlatList                                           |
| ------ | --------------------------------------- | -------------------------------------------------- |
| 렌더링 | 모든 자식 컴포넌트를 한 번에            | 항목이 표시될 때 렌더링                            |
| 단점   | 렌더링 속도가 느리고 메모리 사용량 증가 | 렌더링을 마치지 못하면 빈화면이 보임               |
| 사용   | 데이터 양이 많지 않거나 정해져 있을 때  | 렌더링하는 양이 많고 데이터 양을 예측할 수 없을 때 |
